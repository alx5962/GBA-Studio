# Poachermon sample

This is an original GBA Studio sample project for exercising the GBA build path.

Bundled art is original generated pixel art, not copied from third-party ROMs. The visual target is a lush GBA-era monster-catching RPG mood: green route tiles, tall grass, a small warden office, and compact 16x16 sprites.
For future richer art, prefer CC0 sources and record provenance here before committing binaries.

Useful sources checked:

- Kenney Retro Fantasy Kit: https://www.kenney.nl/assets/retro-fantasy-kit
- OpenGameArt RPG Tilesets Pack: https://opengameart.org/content/rpg-tilesets-pack
- OpenGameArt Top Down Dungeon Pack: https://opengameart.org/content/top-down-dungeon-pack

Current GBA runtime caveat: the native GBA VM is still incomplete, so this project is primarily an end-to-end compiler/build fixture until scene/script playback lands.
